At-1 = At/(1 + BT)
BT = (StPt + Dt - Pt-1)/(Pt-1)
t = (subscript) Day data is pulled from
S = split ratio
P = price
D = dividend value
A = adjusted price

Simple Moving Average
(n + (n - 1) + (n - 2) + ... + (n - y) ) / (y + 1)
n is most recent data.

Exponential Moving Average
SMA: See above
Multiplier: (2 / (y + 2)) = some%
EMA: {Close - EMA(previous day)} * Multiplier + EMA(previous day)
